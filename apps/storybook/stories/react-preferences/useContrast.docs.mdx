# useContrast

A hook that determines whether the user prefers increased contrast. 

This hook returns true if the current device has increased contrast setting enabled. 
The return state will respond to changes in your devices settings and re-render your 
component with the latest setting.

```js
import { useState } from 'react'
import { useContrast } from '@codedazur/react-preferences'

export function Demo() {
  const { prefersMore, prefersLess, prefersCustom } = useContrast();

  return (
    <p>User prefers more contrast: {String(prefersMore)}</p>
    <p>User prefers less contrast: {String(prefersLess)}</p>
    <p>User prefers custom contrast: {String(prefersCustom)}</p>
  )
}
```

## Type Declarations

```js
declare function useContrast(): boolean;
````
