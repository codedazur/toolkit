# useScroll

Returns the current scroll position and progress of an element and exposes
controls over that elements scroll position.

A `useProgress` hook is also returned to opt-in to re-renders when the element's
scroll position changes.

# Usage

```tsx
const { setPosition, useProgress } = useScroll();
const { position } = useProgress();

// scroll down by 100 px on first render
useEffect(() => {
  setPosition({ y: 100 });
}, []);

// will log 0 on first render
// will log 100 on second render
useEffect(() => {
  console.log(position);
}, [position]);
```
